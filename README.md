Como Executar

### Pré-requisitos
- Python 3.8+
- Pacotes necessários: `pillow` (para a versão gráfica)

Instale as dependências:
```bash
pip install pillow

Versão gráfica
python interfaces/grafica/main.py

Versão Terminal
python interfaces/terminal/main.py
